## Authentication

All API endpoints are authenticated using Bearer tokens and picked up from the specification file. To authenticate against the ``Predictor``'s API manager, use the following schema:

```json
"headers": [
  {
    "Ocp-Apim-Subscription-Key": <<YOUR API KEY>>
  }
]
```
